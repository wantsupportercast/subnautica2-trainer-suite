## What Makes This Trainer Unique
Unlike traditional trainers that rely on external overlays or DLL injections, this suite integrates directly with *Subnautica 2*’s memory structures using a lightweight, signature-based scanner. The result is a tool that:
- **Runs alongside the game** without requiring administrator privileges or kernel-level access.
- **Updates automatically** for new game versions (within 48 hours of patches).
- **Preserves achievements** by default, with an optional "sandbox mode" for unrestricted play.
- **Includes a built-in teleporter** with pre-mapped locations (e.g., Alien Base, Primary Containment Facility).

> [!NOTE]
> The trainer does not modify game files. All changes are applied in-memory and reset when the game closes.

## Common Pitfalls
> [!WARNING]
> **Using the trainer with cloud saves (e.g., Steam Cloud) may cause conflicts**. Disable cloud sync before making changes to avoid corruption.

- **Game Updates**: If *Subnautica 2* receives a patch, the trainer may temporarily lose functionality until it is updated. Check the repository for updates.
- **Anti-Cheat**: While the trainer avoids kernel-level hooks, some anti-cheat systems (e.g., EAC) may flag it. Use at your own risk in multiplayer or competitive modes.
- **Overlapping Trainers**: Running multiple trainers simultaneously can cause crashes. Close other tools before using this suite.
- **Save Corruption**: Avoid modifying values during critical game events (e.g., base construction, vehicle piloting). Save manually before and after using the trainer.

## FAQ
### Does this work with *Subnautica: Below Zero*?
No. This trainer is designed exclusively for *Subnautica 2* (2026 release).

### Can I use this in multiplayer?
The trainer is intended for single-player use. Multiplayer compatibility is not supported and may result in bans or desyncs.

### How do I reset changes?
Close the trainer and restart the game. All in-memory modifications are reverted.

### The trainer doesn’t detect the game. What do I do?
- Ensure *Subnautica 2* is running (not in the launcher).
- Restart both the trainer and the game.
- Check for updates to the trainer if the game was recently patched.

> [!TIP]
> If the trainer fails to attach, try running it as administrator *once* to rule out permission issues.
